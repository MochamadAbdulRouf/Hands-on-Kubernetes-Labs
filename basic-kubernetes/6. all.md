# MENAMPILKAN SEMUA RESOURCE DI KUBERNETES

1. Command Pertama
```bash
kubectl get all
```

2. Command Kedua
```bash
kubectl get all --namespace namanamespace
```

# MENGHAPUS SEMUA RESOURCE

1. Command Pertama
```bash
kubectl delete all --all
```

2. Command Kedua
```bash
kubectl delete all --all --namespace namanamespace
```
